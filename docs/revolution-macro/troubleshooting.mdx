# Revolution Macro Troubleshooting

Find solutions to common Revolution Macro issues collected from the community.

## Start Here

Follow this decision flow to find your fix fast.

### Step 1: What OS are you on?

<Accordion title="Windows">
Go to **Step 2** below.
</Accordion>

<Accordion title="Mac">
Go straight to **[Mac Issues](/revolution-macro/mac-issues)** — most Mac problems are permission or compatibility related.
</Accordion>

### Step 2: Is the macro opening?

<Accordion title="No — Macro wont open or crashes on launch">
- **"Application has crashed" / OS-level error** — Try running as Administrator, make sure the macro is extracted from the zip
- **"Failed to initialize log file"** — [See fix](/revolution-macro/common-errors) — macro .exe must be inside a folder
- **Antivirus blocking it** — Temporarily disable antivirus or add an exclusion
- **"Connecting to Backend" stuck forever** — [See fix](/revolution-macro/connection-issues) — reinstall and make sure macro is extracted
</Accordion>

<Accordion title="Yes — Macro opens but something else is wrong">
Go to **Step 3** below.
</Accordion>

### Step 3: What kind of problem?

<Accordion title="Connection / Loading issues">
- **Stuck at "game loading" forever** — [See fix](/revolution-macro/connection-issues) — increase load timeout, try Legacy capture mode
- **Macro keeps rejoining Roblox** — [See fix](/revolution-macro/connection-issues) — check display scale (100%), language (English), disable Auto-HDR
- **TCP error / failed to download DLLs (Russia)** — [See fix](/revolution-macro/connection-issues) — use zapret
- **Failed to open Roblox** — [See fix](/revolution-macro/connection-issues) — run as admin, check Roblox install
</Accordion>

<Accordion title="Gameplay / Movement issues">
- **Character stuttering / walking in circles** — [See fix](/revolution-macro/common-errors) — set camera to Classic, turn off performance stats
- **Wrong field (e.g. goes to sunflower instead of pine tree)** — [See fix](/revolution-macro/common-errors) — reset pattern settings
- **AI gathering walks off field** — [See fix](/revolution-macro/common-errors) — enable High Alignment, check speed settings
- **Macro doesnt find hive / resets when bag full** — [See fix](/revolution-macro/common-errors) — set Hive to Detect in settings
- **Quest menu not showing** — [See fix](/revolution-macro/common-errors) — disable RBC Gather
- **Dipper not collecting** — [See fix](/revolution-macro/common-errors) — verify dipper is equipped in-game
</Accordion>

<Accordion title="RDP-specific issues">
- **Cant harvest when mouse is outside RDP** — [See fix](/revolution-macro/rdp-issues) — known RDP limitation
- **Model load failure on RDP wrapper** — [See fix](/revolution-macro/rdp-issues) — try Legacy capture mode
</Accordion>
