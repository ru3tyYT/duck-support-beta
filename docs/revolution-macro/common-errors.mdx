# Common Errors & Fixes

Known solutions for common Revolution Macro errors.

## Failed to Initialize Log File

> [!WARNING]
> ![Error image](https://github.com/user-attachments/assets/65716114-5ca2-4dcd-87d8-9726cfdfae33)

### Solution

1. Open File Explorer
2. Locate Revolution Macro
3. Ensure You Extracted The Macro
4. Make Sure Macro is Located in Downloads Section or Similar
5. **Revolution Macro .exe MUST be in a folder to save settings, logs etc.**

---

## Macro Not Detecting Roblox

### Solution

1. Change the capture mode to **Legacy** in Settings > Capture
2. Make sure Revolution GUI is near the right side of the screen (if not, macro can have problems with detecting Roblox GUI)
3. In Settings > Misc, change load timeout to a higher value

---

## Stuttering Character Movement

Macro keeps traveling in circles, takes half steps, or fails to detect red cannon.

### Solution

#### Roblox Settings (Main Fixes)
1. In Roblox settings, verify your **Camera Mode** is set to **Classic/Default**
2. Make sure **performance stats** in Roblox are turned off (confuses AI)

#### Revolution Settings
1. Check that your speed in BSS is the same as it is in macro
2. Verify that **Speed Buffs aren't Active**
3. Ensure in Revolution: Settings > Game > Joins > **Hive is set to Detect**
4. If you have lower FPS (17-40), enable **High Alignment** level in Macro: Settings > Game

---

## Quest Menu Not Showing

Macro not showing quest menu.

### Solution

You accidentally enabled **RBC Gather**. Disable it in your settings.

---

## Wrong Field Location

Macro takes me to sunflower instead of pine tree, or goes to the wrong field.

### Solution

1. Try resetting your pattern settings
2. Ensure field coordinates are correct in your configuration
3. Try a different pattern

---

## Pattern Execution Error

### Solution

1. Check your pattern configuration
2. Verify all required fields are accessible
3. Try resetting patterns to default

---

## AI Gathering Walks Off Field

While AI gathering, character walks off the field even with supreme and correct placement.

### Solution

1. Check Roblox settings - ensure Camera Mode is set to Classic/Default
2. Turn off performance stats in Roblox
3. Verify speed settings match between BSS and macro
4. Enable High Alignment in macro settings

---

## Dipper Not Collecting

Dipper doesn't collect even though use dipper option is turned on.

### Solution

1. Verify dipper is equipped in-game
2. Check that the dipper is in your inventory
3. Ensure Use Dipper is enabled in macro settings

---

## Macro Doesnt Find My Hive

Macro goes to rose field and resets when bag is full, or goes into spider and loses items.

### Solution

1. Ensure Hive is set to **Detect** in Settings > Game > Joins
2. Verify your hive is visible on screen when the macro starts
3. Check that your hive location is correct in settings
