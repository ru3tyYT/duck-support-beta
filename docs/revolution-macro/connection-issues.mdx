# Connection Issues

Solutions for connection-related problems with Revolution Macro.

## Infinite "Connecting to Backend" Loading Screen

### Solution

1. Ensure Macro is Extracted (not running from zip)
2. If Macro is Extracted, **Reinstall** and It Should Self-Repair
3. If Still Doesn't Work, Try Moving The Macro GUI Menu Around And Restart

---

## Macro Keeps Rejoining Roblox

### Solution

1. Make sure your **monitor's scale is set to 100%** in Display Settings
2. Make sure your **language is set to English** on the Roblox website
3. Make sure **Auto-HDR is disabled** if your monitor supports it
4. Make sure your **text size is set to zero** in Roblox settings (should show no white bars)
5. Disable any custom fonts or other Bloxstrap settings
6. Go to Settings > Macro > Capture and change Capture Mode to **"Legacy Monitor"**
7. Close and re-open the macro
8. Make sure the macro window is placed on the **bottom right** of your screen

---

## Russia Error (TCP Error, Failed to Download DLLs)

> [!WARNING]
> This error occurs due to network restrictions.

### Solution

1. Download **zapret-discord-youtube** from: https://github.com/Flowseal/zapret-discord-youtube/releases/latest
2. Extract zip file into folder
3. Go to **lists** folder > open **list-general.txt**
4. Add `revolutionmacro.com` to the list and save file
5. Run **zapret**
6. Run macro with zapret running in background

---

## Stuck at Game Load Status

The status after joining the game is still "game loading".

### Solution

1. Check your internet connection
2. Try changing capture mode to Legacy
3. Increase load timeout in Settings > Misc
4. Make sure Roblox is fully loaded before starting the macro

---

## Game Loading Infinite Bug

Macro says "game loading" for 5 mins then restarts Roblox and keeps going until stopped.

### Solution

1. Try running Roblox as administrator
2. Check if Roblox is updated
3. Try a different Roblox server
4. Verify your network connection is stable

---

## Failed to Open Roblox

### Solution

1. Try running Revolution Macro as administrator
2. Verify Roblox is installed correctly
3. Try changing capture mode to Legacy
4. Check that Roblox is not set to run in compatibility mode
