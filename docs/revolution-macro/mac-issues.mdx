# Mac Issues

Troubleshooting for Revolution Macro on macOS.

> [!WARNING]
> **Mac Support Note** — Due to macOS restrictions, the macro may be more buggy than on Windows. Mac support may be in beta.

---

## Macro Doesnt Open (Mac)

When opening the macro, it detects it as malware so user went to settings and opened it, then nothing happened.

### Solution

1. Check Security & Privacy settings in System Preferences
2. Allow the macro to run despite security warnings
3. Try running the macro as administrator or with elevated privileges
4. Verify the macro is extracted (not running from zip)

---

## Mac Permission Fix

After updating, the macro says it doesn't have accessibility permissions.

### Solution

1. Open the **Accessibility settings** menu in **System Settings**
2. Click on **Revolution Macro**
3. Scroll to the bottom and press the **minus button**
4. Press the **plus button**
5. Add Revolution Macro from your Applications folder
6. It should work properly after this

---

## Cant Claim Hive on MacBook

### Solution

1. Due to Mac OS restrictions, the macro is much more buggy
2. Try contacting a developer to test and fix this issue
3. Consider using Atlas (alternative) if Mac issues persist
4. Check Mac privacy settings for accessibility permissions

---

## Mac Typing Issue

Users have problems typing in blank spots in Mac macro - they cant paste or type.

### Solution

This is a known issue on Mac. No fix available at this time according to developers.

---

## Planter Studio Not Working on Mac

After downloading the newest version of macro (0.9a), planter studio stopped working.

### Solution

1. Try reinstalling the macro
2. Try running as administrator
3. This issue affects both RDP and regular Mac usage
4. Use manual mode as a workaround for planters
